# MetaFinOps Web

AI-native FinOps Solutions — Official Website

## Overview

MetaFinOps empowers AI-native enterprises to optimize costs, streamline GPU operations, and enhance green computing initiatives. This repository contains the source code for the MetaFinOps marketing website.

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, responsive design
- **Vanilla JavaScript** — No frameworks, zero dependencies
- **Canvas API** — Particle network and spiral animations

## Project Structure

```
metafinops-github/
├── index.html          # Main HTML page
├── css/
│   └── styles.css      # All styles
├── js/
│   ├── particles.js    # Canvas-based animation classes
│   └── main.js         # App initialization, scroll effects, counters
├── assets/             # Images and media (add your own)
└── README.md
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/metafinops-web.git
   ```

2. Open `index.html` in your browser, or serve locally:
   ```bash
   npx serve .
   ```

3. For GitHub Pages, push to `main` branch and enable Pages in repository settings.

## Features

- Fully responsive (mobile, tablet, desktop)
- Animated particle network backgrounds
- Scroll-triggered animations
- Counter animations for metrics
- Mobile navigation menu
- Newsletter subscription form
- Sticky header with blur effect
